## Discort.site

[![semantic-release](https://img.shields.io/badge/%20%20%F0%9F%93%A6%F0%9F%9A%80-semantic--release-e10079.svg)](https://github.com/semantic-release/semantic-release)

[discort.site](https://discort.site) was formerly being used by discord nitro scamers/spammers/hackers you name it whatever you want, I have found it was available for purchase and decided to buy it
and host this website on it to avoid it usage for further scams. 

The website warns whoever tries to open it and then redirects them to the official discord website.
